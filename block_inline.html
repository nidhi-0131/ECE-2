This is a Block Level Element.

Block level elements start on a new line and take up full width available.

Examples include <div>, <h1> to <h6>, <p>, <ul>, <ol>, and <li>.

This is an Inline Element. Hi, everyone! Welcome to zerotofullstack.

Inline Elements do not start on a new line and only take up as much width as necessary.

Examples include <span>, <a>, <img>, <strong>, and <em>.
